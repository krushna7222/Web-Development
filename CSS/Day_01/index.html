Three Ways To Add CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS

CSS (Cascading Style Sheets) is used to style HTML elements on web pages.
There are three primary types of CSS:
Inline CSS: Apply style rules directly to specific elements within the HTML code.
Internal CSS: Add a `` tag in the `` section of the HTML document.
External CSS: Link an external `.css` file to the HTML document
